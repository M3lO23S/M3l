🔒 أداة M3l - الحماية السيبرانية المتكاملة
ما هي الأداة؟
أداة أمنية شاملة تقوم بـ: كشف الأجهزة المتصلة بشبكتك، تحليل عناوين IP، فحص البرامج الخلفية المشبوهة، مسح المنافذ المفتوحة، اكتشاف الاختراقات، ومراقبة حركة الشبكة.

طريقة التشغيل:
bash
python M3l.py
الأوامر المتاحة:
الأمر	الوظيفة
help	عرض جميع الأوامر
full	فحص كامل للنظام والشبكة
network	كشف الأجهزة على الشبكة
processes	عرض العمليات الخلفية
ports 192.168.1.1 1 100	مسح المنافذ من 1 إلى 100
analyze 8.8.8.8	تحليل عنوان IP
intrusions	كشف الاختراقات
monitor 10	مراقبة الشبكة 10 ثواني
report	عرض التقرير الأمني
save	حفظ النتائج
clear	مسح الشاشة
exit	الخروج
تشغيل مباشر من سطر الأوامر:
bash
python M3l.py full                    # فحص كامل
python M3l.py network                 # كشف الأجهزة
python M3l.py analyze 1.1.1.1        # تحليل IP
python M3l.py ports 192.168.1.1 1 100 # مسح منافذ
